NAME

shasum - Print or Check SHA Checksums

SYNOPSIS

  1.  Usage: shasum [OPTION] [FILE]...
  2.     or: shasum [OPTION] --check [FILE]
  3.  Print or check SHA checksums.
  4.  With no FILE, or when FILE is -, read standard input.
  6.   -a, --algorithm    1 (default), 224, 256, 384, 512
  7.   -b, --binary       read files in binary mode (default on DOS/Windows)
  8.   -c, --check        check SHA sums against given list
  9.   -p, --portable     read files in portable mode
  10.                          produces same digest on Windows/Unix/Mac
  11.   -t, --text         read files in text mode (default)
  13.  The following two options are useful only when verifying checksums:
  15.   -s, --status       don't output anything, status code shows success
  16.   -w, --warn         warn about improperly formatted SHA checksum lines
  18.   -h, --help         display this help and exit
  19.   -v, --version      output version information and exit
  21.  The sums are computed as described in FIPS PUB 180-2.  When checking,
  22.  the input should be a former output of this program.  The default mode
  23.  is to print a line with checksum, a character indicating type (`*'
  24.  for binary, `?' for portable, ` ' for text), and name for each FILE.

DESCRIPTION

The shasum script provides the easiest and most convenient way to compute SHA message digests. Rather than writing a program, the user simply feeds data to the script via the command line, and waits for the results to be printed on standard output. Data can be fed to shasum through files, standard input, or both.

The following command shows how easy it is to compute digests for typical inputs such as the NIST test vector "abc":

  1. 	perl -e "print qw(abc)" | shasum

Or, if you want to use SHA-256 instead of the default SHA-1, simply say:

  1. 	perl -e "print qw(abc)" | shasum -a 256

Since shasum uses the same interface employed by the familiar sha1sum program (and its somewhat outmoded anscestor md5sum), you can install this script as a convenient drop-in replacement.

SEE ALSO

shasum is implemented using the Perl module Digest::SHA or Digest::SHA::PurePerl.
